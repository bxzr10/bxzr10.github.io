text styles :

Following the default hierarchy of h1 > h2 > h3 > etc., I chose to vary fonts depending on the heading level and adjust sizes in order to accomodate for that.

To set up a page with the following styles, I use Inconsolata for the body, Press Start 2P for h1, and adjust size, style, and weight for h2, h3, and .caption class that I use with p elements.

This is an H1 Page Heading

h1 {
  font-family: 'Press Start 2P';
  /* font-size: 1em; */
}

This is a H2 Section Heading

h2 {
  /* font-family: Inconsolata; */
  font-size: 2em;
}

This is an H3 Sub-section Heading

h3 {
  /* font-family: Inconsolata; */
  font-size: 1.5em;
}

This is an H4 Sub-section Heading

h4 {
  /* font-family: Inconsolata; */
  font-size: 1.5em;
  font-size: italic;
}

This is just normal body text. Special descriptions or lines can be offset by 15px on the left (margin-left) in order to break up any dense text.

template setup :

Full HTML and CSS template for new pages.

HTML

<link rel="stylesheet" type="text/css" href="./styles.css" />
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inconsolata:wght@400;500;700&family=Inter:wght@400;700&family=Press+Start+2P&display=swap" rel="stylesheet">

CSS

:root {
  --main-bg-color: #203062;
  --main-text-color: #ED8A8D;
}

body {
  font-family: Inconsolata;
  padding: 1em;
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}

h1 {
  font-family: 'Press Start 2P';
}

h2 {
  font-size: 2em;
}

h3 {
  font-size: 1.5em;
}

h4 {
  font-size: 1.5em;
  font-style: italic;
}

a {
  text-decoration: none;
  color: var(--main-text-color);
}

.caption {
  font-size: 1.25em;
  font-weight: bold;
